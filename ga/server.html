一：如何开服
在https://pan.baidu.com/s/1jIAABA2下载到服务端的jar文件后
放在一个文件夹内，新建一个文本文档，在其中输入
@ECHO OFF
java -Xms1g -Xmx1g -jar AABBCCDD.jar
pause

将其中AABBCCDD.jar改为你下载的服务端jar文件名。最好改为全英文。
将文本文档改名为 run.bat,运行run.bat即可开服。
开服完成后立即关闭run.bat，关闭服务器
用记事本打开文件夹下server.properties
找到online-mode=true，
把它改为online-mode=false，保存
这一步的目的是去除服务器的正版验证，
否则你是进不去的。
之后再打开服务器，就可以用对应r195、r196的客户端进入了。


二：如何在把服务器玩成Miteith
算了吧，服务器刷物品是不现实的，
不过改方块硬度倒是可以，需要服务器和客户端修改成相同的数值。

此外，主世界地图是无法用区块覆盖法直接拥有一堆物品的。
不过地下世界倒是可以考虑，前提是你单机的存档种子和
服务器的一致，而且你需要将搬运的东西放到地下世界去。
如果想修改服务器地图种子，可以将种子值填到
server.properties文件中的level-seed= 一项
除了run.bat、服务器.jar和server.properties，
其他文件都删除，重新运行服务器，关闭服务器。然后将你单机存档里的
DIM-2和region里的文件覆盖到服务器的对应文件夹下，重新开服即可。
现在进入服务器，直接找路可以到地下世界，看看你放的东西在不在？


三：在服务端输入
1：服务端传送指令/spreadplayers
示例：
/spreadplayers X坐标 Z坐标 1 5 true 玩家名
/spreadplayers 233 -666 1 5 true akarin_sama
2：服务器设置复活点/spawnpoint
示例：
坐标缺省则为玩家当前位置
/spawnpoint 玩家名 X坐标 Y坐标 Z坐标
/spawnpoint akarin_sama
